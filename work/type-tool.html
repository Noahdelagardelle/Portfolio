Type Tool

One page site dedicated to manipulating typography right in the browser. I wanted to create a tool that would allow me to play with font pairings, font sizes, colors, etc without having to set up a website from scratch to see how it would look in the browser. The result is Type Tool. Featured on One Page Love.

Visit Site

Once you’ve styled the text to your hearts desire you can click the button on the bottom right side of the screen to get access to the CSS. Then simply copy and paste it into your own projects. Quick and easy.

I added a simple tool at the bottom bar that would suggest font pairings. These are based on Google Fonts suggestions.
